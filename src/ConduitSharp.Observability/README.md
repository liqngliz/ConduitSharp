# ConduitSharp.Observability

Observability types and extensions for ConduitSharp.

- Structured logging and correlation IDs
- OpenTelemetry distributed tracing instrumentation
- Prometheus metrics collection

Automatically integrated into [ConduitSharp.Gateway.AspNetCore](https://www.nuget.org/packages/ConduitSharp.Gateway.AspNetCore) — no manual setup needed. Extend with custom metrics or trace enrichment via the plugin system.
