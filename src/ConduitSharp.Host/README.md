# ConduitSharp.Host

Standalone gateway binary — run ConduitSharp as a separate process without embedding.

Load routes from JSON config, add plugins from your `plugins/` folder, and run as a systemd service or container.

For embedded mode, use [ConduitSharp.Gateway.AspNetCore](https://www.nuget.org/packages/ConduitSharp.Gateway.AspNetCore) instead.
